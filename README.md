# CONTRACT ANALYSIS & RISK ASSESSMENT BOT

AN AI-POWERED LEGAL ASSISTANT DESIGNED FOR INDIAN SMES TO UNDERSTAND CONTRACTS, IDENTIFY RISKS, AND MAKE INFORMED BUSINESS DECISIONS.

## PROBLEM STATEMENT
SMALL AND MEDIUM BUSINESSES OFTEN STRUGGLE TO UNDERSTAND COMPLEX LEGAL CONTRACTS. THIS PROJECT HELPS SMES ANALYZE CONTRACTS, HIGHLIGHT RISKY CLAUSES, AND RECEIVE ACTIONABLE INSIGHTS IN SIMPLE LANGUAGE.

## FEATURES
- UPLOAD CONTRACTS (PDF, DOCX, TXT)
- AUTOMATIC CONTRACT TYPE DETECTION
- CLAUSE-LEVEL RISK ANALYSIS (LOW / MEDIUM / HIGH)
- OVERALL CONTRACT RISK SCORE
- TOP RISKY CLAUSES PRIORITIZATION
- PLAIN-ENGLISH CLAUSE EXPLANATIONS
- SAFER ALTERNATIVE CLAUSE SUGGESTIONS
- NEGOTIATION CHECKLIST
- DOWNLOADABLE RISK SUMMARY REPORT
- PRIVACY-FIRST (LOCAL PROCESSING)

## TECH STACK
- PYTHON
- STREAMLIT
- SPACY
- RULE-BASED NLP
- LLM (USED ONLY FOR EXPLANATION LOGIC)
